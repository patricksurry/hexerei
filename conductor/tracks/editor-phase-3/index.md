## Description
This track implements interactive selection and inspection features for the HexMap editor. It bridges the gap between raw rendering and user interaction by adding bidirectional queries to the `MapModel`, a formal `Selection` model, extended hit-testing for edges and vertices, and a highlight rendering layer in the `Scene`.
