The repository is split into several subprojects:
- `hexmap-rfc/`: Format specification, JSON Schema, modular docs (Python).
- `hexmap-core/`: Core API implementation (TypeScript/Node) handling math, topology, pathfinding.
- `hexmap-importer/`: Python computer vision pipeline to detect hex grids in physical maps.
- `hexmap-renderer/`: TypeScript rendering engine converting HexMap data to visual representations (SVG/Canvas/WebGL).
- `docs/`: High-level design documents and RFCs.
