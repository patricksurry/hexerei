# Tech Stack: HexMap RFC

The project utilizes the following technologies:
- **mmark**: Markdown processor for RFC-style documents (built in Go).
- **Python (>=3.10)**: For build scripts and validation.
- **uv**: Python package and environment manager.
- **jsonschema**: Python library for validating map data against a JSON Schema.
- **xml2rfc**: Tool for generating RFC XML and text output.
- **YAML/JSON**: Core serialization formats for map data and examples.
- **Shell (Darwin/Unix)**: For running build and test commands.
